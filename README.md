# ☁️ Serverless Cloud Quiz

**Description:**  
This is a **serverless Cloud Quiz application** built using **AWS S3, Lambda, API Gateway, and DynamoDB**.  
Users can take a **3-question quiz on cloud computing** and instantly see their score.

**Live Demo:**  
[Open Quiz](http://serverless-quiz-steffy.s3-website.ap-south-1.amazonaws.com)

---

## Features

- Fully **serverless architecture**  
- **Instant score calculation** using AWS Lambda  
- **Data stored in DynamoDB**  
- **Attractive and responsive UI** hosted on S3  

---

## Technologies Used

- **AWS S3** – Static Website Hosting  
- **AWS Lambda** – Serverless Function  
- **API Gateway** – HTTP API  
- **DynamoDB** – NoSQL Database  
- **HTML, CSS, JavaScript** – Frontend

## How to Use

1. Open the **Live Demo link**  
2. Select answers for all 3 questions  
3. Click **Submit Quiz** to see your score  
4. Scores are automatically saved in DynamoDB
